# Upcoming events

<p class="cs4all-intro">This page tracks the public event information currently visible on DIKU's CS4ALL and events pages.</p>

## Current status

<div class="cs4all-note">

## How to stay updated

- Check the official DIKU event calendar for new CS4ALL announcements.
- Join one of the CS4ALL mailing lists to hear about community activity directly.
- Return to the official CS4ALL page for updated registration links and event news.

## Typical event formats

Based on DIKU's own description, CS4ALL activities generally fall into two categories:

- Local networking events for historically underrepresented groups at DIKU.
- Open talks and discussion events where everybody is welcome.
